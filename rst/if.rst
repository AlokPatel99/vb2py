==========

General
-------

``If`` statements are translated to their equivalent Python versions.


Default Conversion
------------------

VB::

    If Value = 10 Or Value = 20 Then
        DoSomething
    ElseIf Value = 30 Then
        DoSomethingElse
    Else
        DontDoAnything
    End If


List of Options
---------------

There are no options specific to the ``If`` statement.
